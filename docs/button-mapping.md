# 各畫面按鈕功能對照

本文件整理目前裝置 UI 各畫面的按鈕功能。內容依照目前原始碼中的 `MappedInputManager` 與各 Activity 的 `loop()` 行為整理。

## 全域按鈕對應

- 底部四鍵預設為：返回/取消、確認、左、右。
- 底部四鍵可在設定中重新映射，程式使用 `SETTINGS.frontButtonBack`、`SETTINGS.frontButtonConfirm`、`SETTINGS.frontButtonLeft`、`SETTINGS.frontButtonRight`。
- 側邊 `上`、`下` 固定對應實體側鍵。
- 閱讀翻頁使用 `PageBack`、`PageForward`，也是側鍵，但可透過 `sideButtonLayout` 對調。
- 電源鍵不走重新映射；閱讀中可依 `shortPwrBtn` 設定作為短按翻頁。

## 主畫面

- `上` / `左`：上一個項目。
- `下` / `右`：下一個項目。
- `確認`：開啟目前選到的書或功能。
- `返回`：主畫面本身沒有特別處理。

## SD 檔案總管

- 一般模式下 `左` / `右`：不切換操作列。
- `上` / `下`：移動檔案清單。
- `確認` 短按：開啟目前選到的資料夾或檔案。
- `確認` 長按：顯示上方操作列。
- 操作列顯示時 `左` / `右`：切換操作項目，順序為取消、刪除、複製、剪下、貼上。
- 操作列一次最多顯示三個項目，移到後段項目時會翻到後面的操作。
- 操作列顯示時 `確認`：執行目前選到的操作。
- 操作列中的 `取消`：隱藏操作列。
- `返回` 短按：回上一層資料夾；在根目錄時回主畫面。

## 設定頁

- `上`：上一個設定項。
- `下`：下一個設定項。
- 選到分類列時，`左` / `右`：切換設定分類。
- 選到一般設定項時，`左`：切到前一個設定值。
- 選到一般設定項時，`右`：切到後一個設定值。
- 選到動作項時，`右`：執行該動作，例如藍牙、清除快取、更新或字型選擇。
- `確認` / `返回`：儲存設定並回主畫面。

## 閱讀頁：EPUB / TXT / XTC

- `左` 或 `PageBack`：上一頁。
- `右` 或 `PageForward`：下一頁。
- `確認`：EPUB 開閱讀選單；TXT / XTC 開章節選擇。
- `返回`：回檔案總管。
- `電源` 短按：只有在 `shortPwrBtn` 設定為 `PAGE_TURN` 時作為下一頁。

## 圖片閱讀

一般模式：

- `左` / `上`：上一張。
- `右` / `下`：下一張。
- `確認`：開啟選單。
- `返回`：回檔案總管。

選單模式：

- `左` / `上`：上一項。
- `右` / `下`：下一項。
- `確認`：執行目前選項。
- `返回`：關閉選單。

## EPUB 閱讀選單 / 章節選單

- `上` / `左`：上一項。
- `下` / `右`：下一項。
- `確認`：選擇。
- `返回`：回閱讀頁。

## 百分比跳轉頁

- `左`：小幅降低百分比。
- `右`：小幅提高百分比。
- `上`：大幅提高百分比。
- `下`：大幅降低百分比。
- `確認`：套用。
- `返回`：取消。

## OPDS 瀏覽器

瀏覽狀態：

- `上` / `PageBack`：上一項。
- `下` / `PageForward`：下一項。
- `左`：切到 OPDS 上一頁 feed。
- `右`：切到 OPDS 下一頁 feed。
- `確認`：進入目錄或下載書。
- `返回`：上一層或返回。

錯誤狀態：

- `確認`：重試。
- `返回`：返回。

檢查 Wi-Fi / 載入狀態：

- `返回`：取消或返回。

下載狀態：

- 不接受按鍵輸入。

## Wi-Fi / 網路模式

網路模式選單：

- `上` / `左`：上一項。
- `下` / `右`：下一項。
- `確認`：選擇。
- `返回`：取消。

Web server / Calibre 傳輸頁：

- `返回`：離開或停止目前傳輸頁。

## 鍵盤輸入

- `上` / `下` / `左` / `右`：移動鍵盤游標。
- `確認`：輸入字元，或執行 Shift、空白、退格、OK、QR。
- `返回`：取消輸入。
- QR 畫面時 `返回`：離開 QR 模式並回鍵盤。

## 藍牙設定

主選單：

- `上` / `下`：選擇項目。
- `確認`：開關藍牙或進入裝置清單。
- `返回`：離開藍牙設定。

裝置清單：

- `上` / `下`：選擇裝置或清單操作。
- `左`：回主選單。
- `右`：重新掃描。
- `確認`：連線、刷新或斷線，依目前選到的項目而定。
- `返回`：回主選單。

## 字型選擇

- `上` / `左`：上一項。
- `下` / `右`：下一項。
- `確認`：套用目前字型並離開。
- `返回`：取消並離開。

## KOReader / OPDS 設定

- `上` / `左`：上一項。
- `下` / `右`：下一項。
- `確認`：選擇或編輯目前項目。
- `返回`：返回上一頁。

## KOReader 同步

一般結果選擇畫面：

- `上` / `左`：上一個選項。
- `下` / `右`：下一個選項。
- `確認`：執行目前選項，例如套用遠端進度、上傳本機進度或取消。
- `返回`：取消。

沒有登入資料、同步失敗、上傳完成等狀態：

- `返回`：離開。

沒有遠端進度狀態：

- `確認`：上傳本機進度。
- `返回`：取消。

## OTA 更新

等待確認狀態：

- `確認`：開始更新。
- `返回`：取消並返回。

失敗或沒有更新狀態：

- `返回`：返回。

更新成功後：

- 程式會重開機。

## 清除快取

警告狀態：

- `確認`：開始清除快取。
- `返回`：取消並返回。

完成或失敗狀態：

- `返回`：返回。

## 按鈕重新映射

- 側邊 `上`：重設底部四鍵為預設映射並離開。
- 側邊 `下`：取消，不儲存並離開。
- 底部任一鍵：依畫面提示依序指定為返回、確認、左、右。
- 完成四個角色指定後會儲存並離開。

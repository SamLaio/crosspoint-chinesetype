## 1. 硬件概览

设备使用 Xtink X4 的标准按钮（默认布局与原厂固件相同）：

### 按钮布局
| 位置 | 按钮 |
| --------------- | ---------------------------------------------------- |
| **底部边缘** | **返回 (Back)**、**确认 (Confirm)**、**左 (Left)**、**右 (Right)** |
| **右侧边** | **电源 (Power)**、**音量加 (Vol Up)**、**音量减 (Vol Down)**、**重置 (Reset)** |

按钮布局可以在 **[设置](#35-设置)** 中自定义。

## 2. 电源与启动

### 开机/关机

要开启或关闭设备，**长按电源键约半秒**。
在 **[设置](#35-设置)** 中，您可以配置电源键为短按关机，而非长按。

要重启设备（例如死机或固件更新后），按下并松开重置键，然后迅速长按电源键几秒钟。

### 首次启动

首次开机时，您将进入 **[主屏幕](#31-主屏幕)**。

> [!NOTE] 注意
> 在后续重启时，固件会自动重新打开您最后阅读的书籍。

## 3. 屏幕界面

### 3.1 主屏幕

主屏幕是固件的主要入口。从这里您可以进入最近阅读书籍的 **[阅读模式](#4-阅读模式)**、**[书籍选择](#32-书籍选择)**、**[设置](#35-设置)** 或 **[文件上传](#34-文件上传界面)** 界面。

### 3.2 书籍选择

书籍选择界面充当文件夹和文件浏览器。

* **浏览列表：** 使用 **左**（或 **音量加**）、或 **右**（或 **音量减**）上下移动选择光标。您也可以长按这些按钮整页翻动。
* **打开选择：** 按 **确认** 打开文件夹或阅读选中的书籍。

### 3.3 阅读模式

详见下方的 [阅读模式](#4-阅读模式)。

### 3.4 文件上传界面

文件上传界面允许您上传新的电子书到设备。进入该界面后，会弹出 WiFi 选择对话框，随后您的 X4 将启动一个 Web 服务器。

有关如何连接 Web 服务器和上传文件的更多信息，请参阅 [Web 服务器文档](./docs/webserver.md)。

> [!TIP] 提示
> 高级用户还可以使用 `curl` 通过编程或命令行管理文件。详情请见 [Web 服务器文档](./docs/webserver.md)。

#### 3.4.1 Calibre 无线传输

CrossPoint 支持使用 CrossPoint Reader 设备插件从 Calibre 发送书籍。

1. 在 Calibre 中安装插件：
   - 访问 https://github.com/crosspoint-reader/calibre-plugins/releases 下载最新版的 crosspoint_reader 插件。
   - 下载 zip 文件。
   - 打开 Calibre → 首选项 → 插件 → 从文件加载插件 → 选择 zip 文件。
2. 在设备上：文件传输 → 连接 Calibre → 加入网络。
3. 确保您的电脑在同一 WiFi 网络下。
4. 在 Calibre 中，点击“发送到设备”以传输书籍。

### 3.5 设置

设置界面允许您配置设备行为。您可以调整以下设置：

- **休眠屏幕 (Sleep Screen)**：设备休眠时显示的屏幕：
  - "Dark" (默认) - 默认的深色 Crosspoint Logo 休眠屏
  - "Light" - 同样的默认休眠屏，但在白色背景上
  - "Custom" - 来自 SD 卡的自定义图片；详见下方 [休眠屏幕](#36-休眠屏幕)
  - "Cover" - 书籍封面图（注：此为实验性功能，可能无法按预期工作）
  - "None" - 空白屏幕
  - "Cover + Custom" - 书籍封面图，若无则回退到 "Custom"
- **休眠屏幕封面模式 (Sleep Screen Cover Mode)**：选择 "Cover" 时如何显示封面：
  - "Fit" (默认) - 缩放图片以适应屏幕中心，必要时填充白边
  - "Crop" - 缩放并裁剪图片以尝试填满屏幕（注：实验性功能）
- **休眠屏幕封面滤镜 (Sleep Screen Cover Filter)**：选择 "Cover" 时对封面应用的滤镜：
  - "None" (默认) - 转换为灰度图并原样显示
  - "Contrast" - 显示为黑白图，不进行灰度转换
  - "Inverted" - 反色显示（白变黑，黑变白），不进行灰度转换
- **状态栏 (Status Bar)**：配置阅读时显示的状态栏：
  - "None" - 无状态栏
  - "No Progress" - 显示状态栏但不显示阅读进度
  - "Full w/ Percentage" - 显示状态栏及书籍进度（百分比）
  - "Full w/ Book Bar" - 显示状态栏及书籍进度（进度条）
  - "Book Bar Only" - 仅显示书籍进度（进度条）
  - "Full w/ Chapter Bar" - 显示状态栏及章节进度（进度条）
- **隐藏电池百分比 (Hide Battery %)**：配置在状态栏中何处隐藏电池百分比（电池图标仍会显示）：
  - "Never" - 始终显示电池百分比 (默认)
  - "In Reader" - 除阅读模式外均显示
  - "Always" - 始终隐藏电池百分比
- **额外段落间距 (Extra Paragraph Spacing)**：启用后，段落间会增加垂直间距。禁用后，段落间无垂直间距，但会有首行缩进。(此处已改为手动增加，因为英文这个不适合)
- **文本抗锯齿 (Text Anti-Aliasing)**：是否在阅读模式下对文本边缘进行平滑处理（抗锯齿）。注意这会略微降低翻页速度。
- **短按电源键 (Short Power Button Click)**：控制短按电源键的效果：
  - "Ignore" - 需要长按才能关机
  - "Sleep" - 短按即关机
  - "Page Turn" - 阅读模式下短按翻到下一页；长按关机
- **阅读方向 (Reading Orientation)**：设置阅读 EPUB 时的屏幕方向：
  - "Portrait" (默认) - 标准竖屏
  - "Landscape CW" - 横屏，顺时针旋转
  - "Inverted" - 竖屏，倒置
  - "Landscape CCW" - 横屏，逆时针旋转
- **正面按钮布局 (Front Button Layout)**：配置底部边缘按钮的顺序：
  - 返回, 确认, 左, 右 (默认)
  - 左, 右, 返回, 确认
  - 左, 返回, 确认, 右
  - 返回, 确认, 右, 左
- **侧面按钮布局 (阅读器) (Side Button Layout)**：交换音量加减键的上/下顺序（从 上一页/下一页 变为 下一页/上一页）。此更改仅在阅读时生效。
- **长按跳章 (Long-press Chapter Skip)**：设置长按翻页键是否跳转到下一/上一章。
  - "Chapter Skip" (默认) - 长按跳转章节
  - "Page Scroll" - 长按整页滚动
- **阅读器字体家族 (Reader Font Family)**：选择阅读使用的字体：
  - "Bookerly" (默认) - Amazon 的阅读字体
  - "Noto Sans" - Google 的无衬线字体
  - "Open Dyslexic" - 专为阅读障碍者设计的字体
- **阅读器字体大小 (Reader Font Size)**：调整文本大小；选项有 "Small", "Medium", "Large", "X Large"。
- **阅读器行间距 (Reader Line Spacing)**：调整行间距；选项有 "Tight", "Normal", "Wide"。
- **阅读器屏幕边距 (Reader Screen Margin)**：控制阅读模式下的屏幕边距，范围 5 到 40 像素，步长 5 像素。
- **阅读器段落对齐 (Reader Paragraph Alignment)**：设置段落对齐方式；选项有 "Justified" (默认), "Left", "Center", "Right"。
- **休眠时间 (Time to Sleep)**：设置设备自动休眠前的无操作时长。
- **刷新频率 (Refresh Frequency)**：设置阅读时全屏刷新的频率以减少残影。
- **阳光褪色修复 (Sunlight Fading Fix)**：配置是否启用软件修复，解决白色 X4 机型在直射阳光下可能褪色的问题
  - "OFF" (默认) - 禁用修复
  - "ON" - 启用修复
- **OPDS 浏览器 (OPDS Browser)**：配置 OPDS 服务器设置以浏览和下载书籍。设置服务器 URL（对于 Calibre Content Server，需在末尾添加 `/opds`），并可配置用户名和密码以进行认证。注意：仅支持 HTTP Basic 认证。如果使用启用了认证的 Calibre Content Server，必须将其设置为 Basic 认证，而非默认的 Digest 认证。
- **检查更新 (Check for updates)**：通过 WiFi 检查固件更新。

### 3.6 休眠屏幕

您可以通过在 SD 卡的特定位置放置自定义图片来定制休眠屏幕：

- **单张图片：** 在根目录放置名为 `sleep.bmp` 的文件。（有sleep/*bmp的时候不起效）
- **多张图片：** 在 SD 卡根目录创建 `sleep` 文件夹，并在其中放入任意数量的 `.bmp` 图片。如果找到此文件夹中的图片，它们将优先于 `sleep.bmp` 文件，且每次设备休眠时会随机选择一张。

> [!NOTE] 注意
> 您需要在 **设置** 中将 **休眠屏幕** 设置为 **Custom** 才能使用这些图片。

> [!TIP] 提示
> 为了获得最佳效果：
> - 使用未压缩的 24 位色深 BMP 文件
> - 使用 480x800 像素的分辨率以匹配设备屏幕分辨率。

## 4. 阅读模式

打开书籍后，按钮布局会发生变化以方便阅读。

### 翻页
| 动作 | 按钮 |
| ----------------- | ------------------------------------ |
| **上一页** | 按 **左** _或_ **音量加** |
| **下一页** | 按 **右** _或_ **音量减** |

音量（侧面）按钮的功能可以在 **[设置](#35-设置)** 中交换。

如果 **短按电源键** 设置为 "Page Turn"，您也可以通过短按电源键翻到下一页。

### 章节导航
* **下一章：** **长按** **右**（或 **音量减**）按钮片刻，然后松开。
* **上一章：** **长按** **左**（或 **音量加**）按钮片刻，然后松开。

此功能可以在 **[设置](#35-设置)** 中禁用，以避免误触切换章节。

### 系统导航
* **返回书籍选择：** 按 **返回** 关闭书籍并返回 **[书籍选择](#32-书籍选择)** 界面。
* **返回主页：** **长按** **返回** 按钮关闭书籍并返回 **[主页](#31-主屏幕)** 界面。
* **章节菜单：** 按 **确认** 打开 **[目录/章节选择](#5-章节选择界面)**。

### 支持的语言

CrossPoint 使用以下 Unicode 字符块渲染文本，支持多种语言：

*   **拉丁脚本 (基本、补充、扩展-A)：** 覆盖英语、德语、法语、西班牙语、葡萄牙语、意大利语、荷兰语、瑞典语、挪威语、丹麦语、芬兰语、波兰语、捷克语、匈牙利语、罗马尼亚语、斯洛伐克语、斯洛文尼亚语、土耳其语等。
*   **西里尔脚本 (标准和扩展)：** 覆盖俄语、乌克兰语、白俄罗斯语、保加利亚语、塞尔维亚语、马其顿语、哈萨克语、吉尔吉斯语、蒙古语等。

**不支持的语言：** 中文、日语、韩语、越南语、希伯来语、阿拉伯语、希腊语和波斯语。

## 5. 章节选择界面

在书籍内按 **确认** 即可访问。

1.  使用 **左**（或 **音量加**）、或 **右**（或 **音量减**）高亮显示所需章节。
2.  按 **确认** 跳转到该章节。
3.  *或者，按 **返回** 取消并返回当前页面。*

## 6. 当前限制与路线图

请注意，此固件目前处于积极开发中。以下功能 **尚未支持**，但计划在未来更新中添加：

* **图片：** 电子书中的嵌入图片将无法渲染。

## 7. 故障排除与退出启动循环

如果在使用 Crosspoint 时遇到问题或崩溃，请随时提交问题工单并附上串口监视器日志。可以通过将设备连接到电脑并启动串口监视器来获取日志。可以使用 [Serial Monitor](https://www.serialmonitor.org/) 或以下命令：

```bash
pio device monitor
```

如果设备陷入启动循环（Bootloop），请按一下并松开重置键。然后，**长按** 配置的 **返回键** 和 **电源键** 以启动到主屏幕。

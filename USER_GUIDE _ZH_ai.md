## 1. 硬體概覽

裝置使用 Xtink X4 的標準按鈕（預設佈局與原廠韌體相同）：

### 按鈕佈局
| 位置 | 按鈕 |
| --------------- | ---------------------------------------------------- |
| **底部邊緣** | **返回 (Back)**、**確認 (Confirm)**、**左 (Left)**、**右 (Right)** |
| **右側邊** | **電源 (Power)**、**音量加 (Vol Up)**、**音量減 (Vol Down)**、**重置 (Reset)** |

按鈕佈局可以在 **[設定](#35-設定)** 中自定義。

---

## 2. 電源與啟動

### 開機/關機

要開啟或關閉裝置，**長按電源鍵約半秒**。
在 **[設定](#35-設定)** 中，您可以配置電源鍵為短按關機，而非長按。

要重啟裝置（例如宕機或韌體更新後），按下並鬆開重置鍵，然後迅速長按電源鍵幾秒鐘。

### 首次啟動

首次開機時，您將進入 **[主螢幕](#31-主螢幕)**。

> [!NOTE] 注意
> 在後續重啟時，韌體會自動重新開啟您最後閱讀的書籍。

## 3. 螢幕介面

### 3.1 主螢幕

主螢幕是韌體的主要入口。從這裡您可以進入最近閱讀書籍的 **[閱讀模式](#4-閱讀模式)**、**[書籍選擇](#32-書籍選擇)**、**[設定](#35-設定)** 或 **[檔案上傳](#34-檔案上傳介面)** 介面。

### 3.2 書籍選擇

書籍選擇介面充當資料夾和檔案瀏覽器。

* **瀏覽列表：** 使用 **左**（或 **音量加**）、或 **右**（或 **音量減**）上下移動選擇游標。您也可以長按這些按鈕整頁翻動。
* **開啟選擇：** 按 **確認** 開啟資料夾或閱讀選中的書籍。

### 3.3 閱讀模式

詳見下方的 [閱讀模式](#4-閱讀模式)。

### 3.4 檔案上傳介面

檔案上傳介面允許您上傳新的電子書到裝置。進入該介面後，會彈出 WiFi 選擇對話方塊，隨後您的 X4 將啟動一個 Web 伺服器。

有關如何連線 Web 伺服器和上傳檔案的更多資訊，請參閱 [Web 伺服器文件](./docs/webserver.md)。

> [!TIP] 提示
> 高階使用者還可以使用 `curl` 透過程式設計或命令列管理檔案。詳情請見 [Web 伺服器文件](./docs/webserver.md)。

#### 3.4.1 Calibre 無線傳輸

CrossPoint 支援使用 CrossPoint Reader 裝置外掛從 Calibre 傳送書籍。

1. 在 Calibre 中安裝外掛：
   - 訪問 https://github.com/crosspoint-reader/calibre-plugins/releases 下載最新版的 crosspoint_reader 外掛。
   - 下載 zip 檔案。
   - 開啟 Calibre → 首選項 → 外掛 → 從檔案載入外掛 → 選擇 zip 檔案。
2. 在裝置上：檔案傳輸 → 連線 Calibre → 加入網路。
3. 確保您的電腦在同一 WiFi 網路下。
4. 在 Calibre 中，點選“傳送到裝置”以傳輸書籍。

### 3.5 設定

設定介面允許您配置裝置行為。您可以調整以下設定：

- **休眠螢幕 (Sleep Screen)**：裝置休眠時顯示的螢幕：
  - "Dark" (預設) - 預設的深色 Crosspoint Logo 休眠屏
  - "Light" - 同樣的預設休眠屏，但在白色背景上
  - "Custom" - 來自 SD 卡的自定義圖片；詳見下方 [休眠螢幕](#36-休眠螢幕)
  - "Cover" - 書籍封面圖（注：此為實驗性功能，可能無法按預期工作）
  - "None" - 空白螢幕
  - "Cover + Custom" - 書籍封面圖，若無則回退到 "Custom"
- **休眠螢幕封面模式 (Sleep Screen Cover Mode)**：選擇 "Cover" 時如何顯示封面：
  - "Fit" (預設) - 縮放圖片以適應螢幕中心，必要時填充白邊
  - "Crop" - 縮放並裁剪圖片以嘗試填滿螢幕（注：實驗性功能）
- **休眠螢幕封面濾鏡 (Sleep Screen Cover Filter)**：選擇 "Cover" 時對封面應用的濾鏡：
  - "None" (預設) - 轉換為灰度圖並原樣顯示
  - "Contrast" - 顯示為黑白圖，不進行灰度轉換
  - "Inverted" - 反色顯示（白變黑，黑變白），不進行灰度轉換
- **狀態列 (Status Bar)**：配置閱讀時顯示的狀態列：
  - "None" - 無狀態列
  - "No Progress" - 顯示狀態列但不顯示閱讀進度
  - "Full w/ Percentage" - 顯示狀態列及書籍進度（百分比）
  - "Full w/ Book Bar" - 顯示狀態列及書籍進度（進度條）
  - "Book Bar Only" - 僅顯示書籍進度（進度條）
  - "Full w/ Chapter Bar" - 顯示狀態列及章節進度（進度條）
- **隱藏電池百分比 (Hide Battery %)**：配置在狀態列中何處隱藏電池百分比（電池圖示仍會顯示）：
  - "Never" - 始終顯示電池百分比 (預設)
  - "In Reader" - 除閱讀模式外均顯示
  - "Always" - 始終隱藏電池百分比
- **額外段落間距 (Extra Paragraph Spacing)**：啟用後，段落間會增加垂直間距。禁用後，段落間無垂直間距，但會有首行縮排。(此處已改為手動增加，因為英文這個不適合)
- **文字抗鋸齒 (Text Anti-Aliasing)**：是否在閱讀模式下對文字邊緣進行平滑處理（抗鋸齒）。注意這會略微降低翻頁速度。
- **短按電源鍵 (Short Power Button Click)**：控制短按電源鍵的效果：
  - "Ignore" - 需要長按才能關機
  - "Sleep" - 短按即關機
  - "Page Turn" - 閱讀模式下短按翻到下一頁；長按關機
- **閱讀方向 (Reading Orientation)**：設定閱讀 EPUB 時的螢幕方向：
  - "Portrait" (預設) - 標準豎屏
  - "Landscape CW" - 橫屏，順時針旋轉
  - "Inverted" - 豎屏，倒置
  - "Landscape CCW" - 橫屏，逆時針旋轉
- **正面按鈕佈局 (Front Button Layout)**：配置底部邊緣按鈕的順序：
  - 返回, 確認, 左, 右 (預設)
  - 左, 右, 返回, 確認
  - 左, 返回, 確認, 右
  - 返回, 確認, 右, 左
- **側面按鈕佈局 (閱讀器) (Side Button Layout)**：交換音量加減鍵的上/下順序（從 上一頁/下一頁 變為 下一頁/上一頁）。此更改僅在閱讀時生效。
- **長按跳章 (Long-press Chapter Skip)**：設定長按翻頁鍵是否跳轉到下一/上一章。
  - "Chapter Skip" (預設) - 長按跳轉章節
  - "Page Scroll" - 長按整頁滾動
- **閱讀器字型家族 (Reader Font Family)**：選擇閱讀使用的字型：
  - "Bookerly" (預設) - Amazon 的閱讀字型
  - "Noto Sans" - Google 的無襯線字型
  - "Open Dyslexic" - 專為閱讀障礙者設計的字型
- **閱讀器字型大小 (Reader Font Size)**：調整文字大小；選項有 "Small", "Medium", "Large", "X Large"。
- **閱讀器行間距 (Reader Line Spacing)**：調整行間距；選項有 "Tight", "Normal", "Wide"。
- **閱讀器螢幕邊距 (Reader Screen Margin)**：控制閱讀模式下的螢幕邊距，範圍 5 到 40 畫素，步長 5 畫素。
- **閱讀器段落對齊 (Reader Paragraph Alignment)**：設定段落對齊方式；選項有 "Justified" (預設), "Left", "Center", "Right"。
- **休眠時間 (Time to Sleep)**：設定裝置自動休眠前的無操作時長。
- **重新整理頻率 (Refresh Frequency)**：設定閱讀時全屏重新整理的頻率以減少殘影。
- **陽光褪色修復 (Sunlight Fading Fix)**：配置是否啟用軟體修復，解決白色 X4 機型在直射陽光下可能褪色的問題
  - "OFF" (預設) - 禁用修復
  - "ON" - 啟用修復
- **OPDS 瀏覽器 (OPDS Browser)**：配置 OPDS 伺服器設定以瀏覽和下載書籍。設定伺服器 URL（對於 Calibre Content Server，需在末尾新增 `/opds`），並可配置使用者名稱和密碼以進行認證。注意：僅支援 HTTP Basic 認證。如果使用啟用了認證的 Calibre Content Server，必須將其設定為 Basic 認證，而非預設的 Digest 認證。
- **檢查更新 (Check for updates)**：透過 WiFi 檢查韌體更新。

### 3.6 休眠螢幕

您可以透過在 SD 卡的特定位置放置自定義圖片來定製休眠螢幕：

- **單張圖片：** 在根目錄放置名為 `sleep.bmp` 的檔案。（有sleep/*bmp的時候不起效）
- **多張圖片：** 在 SD 卡根目錄建立 `sleep` 資料夾，並在其中放入任意數量的 `.bmp` 圖片。如果找到此資料夾中的圖片，它們將優先於 `sleep.bmp` 檔案，且每次裝置休眠時會隨機選擇一張。

> [!NOTE] 注意
> 您需要在 **設定** 中將 **休眠螢幕** 設定為 **Custom** 才能使用這些圖片。

> [!TIP] 提示
> 為了獲得最佳效果：
> - 使用未壓縮的 24 位色深 BMP 檔案
> - 使用 480x800 畫素的解析度以匹配裝置螢幕解析度。

## 4. 閱讀模式

開啟書籍後，按鈕佈局會發生變化以方便閱讀。

### 翻頁
| 動作 | 按鈕 |
| ----------------- | ------------------------------------ |
| **上一頁** | 按 **左** _或_ **音量加** |
| **下一頁** | 按 **右** _或_ **音量減** |

音量（側面）按鈕的功能可以在 **[設定](#35-設定)** 中交換。

如果 **短按電源鍵** 設定為 "Page Turn"，您也可以透過短按電源鍵翻到下一頁。

### 章節導航
* **下一章：** **長按** **右**（或 **音量減**）按鈕片刻，然後鬆開。
* **上一章：** **長按** **左**（或 **音量加**）按鈕片刻，然後鬆開。

此功能可以在 **[設定](#35-設定)** 中禁用，以避免誤觸切換章節。

### 系統導航
* **返回書籍選擇：** 按 **返回** 關閉書籍並返回 **[書籍選擇](#32-書籍選擇)** 介面。
* **返回主頁：** **長按** **返回** 按鈕關閉書籍並返回 **[主頁](#31-主螢幕)** 介面。
* **章節選單：** 按 **確認** 開啟 **[目錄/章節選擇](#5-章節選擇介面)**。

### 支援的語言

CrossPoint 使用以下 Unicode 字元塊渲染文字，支援多種語言：

*   **拉丁指令碼 (基本、補充、擴充套件-A)：** 覆蓋英語、德語、法語、西班牙語、葡萄牙語、義大利語、荷蘭語、瑞典語、挪威語、丹麥語、芬蘭語、波蘭語、捷克語、匈牙利語、羅馬尼亞語、斯洛伐克語、斯洛維尼亞語、土耳其語等。
*   **西里爾指令碼 (標準和擴充套件)：** 覆蓋俄語、烏克蘭語、白俄羅斯語、保加利亞語、塞爾維亞語、馬其頓語、哈薩克語、吉爾吉斯語、蒙古語等。

**不支援的語言：** 中文、日語、韓語、越南語、希伯來語、阿拉伯語、希臘語和波斯語。

## 5. 章節選擇介面

在書籍內按 **確認** 即可訪問。

1.  使用 **左**（或 **音量加**）、或 **右**（或 **音量減**）高亮顯示所需章節。
2.  按 **確認** 跳轉到該章節。
3.  *或者，按 **返回** 取消並返回當前頁面。*

---

## 6. 當前限制與路線圖

請注意，此韌體目前處於積極開發中。以下功能 **尚未支援**，但計劃在未來更新中新增：

* **圖片：** 電子書中的嵌入圖片將無法渲染。

## 7. 故障排除與退出啟動迴圈

如果在使用 Crosspoint 時遇到問題或崩潰，請隨時提交問題工單並附上串列埠監視器日誌。可以透過將裝置連線到電腦並啟動串列埠監視器來獲取日誌。可以使用 [Serial Monitor](https://www.serialmonitor.org/) 或以下命令：

```bash
pio device monitor
```

如果裝置陷入啟動迴圈（Bootloop），請按一下並鬆開重置鍵。然後，**長按** 配置的 **返回鍵** 和 **電源鍵** 以啟動到主螢幕。
